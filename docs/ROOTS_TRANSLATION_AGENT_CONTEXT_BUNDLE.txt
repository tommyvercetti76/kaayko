# ROOTS Translation Pipeline - System Map

## Purpose
This document captures what has been built for the ROOTS local translation pipeline, including:
- Full end-to-end behavior
- Complete file map (UI + API + storage touchpoints)
- Dependency loops (runtime and code-level)
- Operational constraints and environment requirements

Scope here is the translation workflow used by:
- Individual translation/review
- Batch translation (max 25)
- Review queue and approvals
- Local Ollama runtime diagnostics

---

## What Is Built (Current State)

### 1) Translation UI (local-first admin)
- Static admin page with tabs:
  - Individual
  - Batch
  - Review Queue
- Local mode support:
  - API override via `?api=http://localhost:8081`
  - Dev auth bypass via `?dev=1` (token=`Bearer dev`)
- Runtime diagnostics panel:
  - shows model/base URL/reachability
- Center-panel readability mode:
  - Source + translated question summary in center
  - Full editable translated fields in right panel

### 2) Translation API
- Translation endpoints under `.../api/v1/roots/prototype-admin`
- Runtime status endpoint
- Single question translate endpoint
- Review/save endpoint
- Batch create + progress/list endpoints

### 3) Translation service
- Source question retrieval by source type (`parent`, `teacher`, `child`, `generated`)
- Local model field-by-field translation strategy
- Optional second-stage cleanup pass (another LLM)
- Translation persistence in Firestore
- Tag memory persistence for consistency

### 4) Batch processing
- Max 25 questions per batch
- Max 2 concurrent active batches
- In-service loop translation with progress updates
- UI batch list with polling + result drilldown

## Runtime Topology

```text
Browser (localhost:8090)
  -> Static UI page (Kaayko_v6 HTML)
  -> Calls API (localhost:8081)
      -> ROOTS route /prototype-admin
          -> question-translation.service
              -> Ollama local endpoint (127.0.0.1:11434)
              -> Firestore (coolschools-72426)
```

Notes:
- Local browser page is served via simple local HTTP server to avoid HTTPS->HTTP mixed-content restrictions.
- API CORS allows `http://localhost:8090` in dev.

## End-to-End Dependency Loops

## Loop A: Individual Translate
1. UI selects source+locale+question.
2. UI calls `GET /translations/questions/:source/:questionId`.
3. API loads source question + existing translations.
4. User clicks Translate.
5. UI calls `GET /translations/runtime-status`.
6. UI calls `POST /translations/questions/:source/:questionId/translate`.
7. Service translates fields via Ollama (`/api/generate`).
8. Optional cleanup pass via second model (`/api/chat`) if configured.
9. Service writes translation document to Firestore.
10. UI reloads question bundle and renders translated output.

## Loop B: Review/Save
1. User edits translated fields in right panel.
2. UI calls `POST /translations/questions/:source/:questionId/review` with status.
3. Service validates answer count and writes reviewed translation.
4. Service updates tag memory.
5. UI updates list badges and center compare panel.

## Loop C: Batch Translate
1. User selects up to 25 IDs in Batch tab.
2. UI calls `POST /translations/batches`.
3. Service creates batch doc (`processing`) and loops question IDs.
4. Each ID calls same `translateSingle` path as individual translation.
5. Batch doc increments success/failed counters.
6. Batch doc ends as `completed` or `completed_with_errors`.
7. UI polls `GET /translations/batches` until done.
8. User clicks View Results -> Review Queue filtered to source/locale/status.

## Loop D: Runtime Diagnostics
1. UI calls `GET /translations/runtime-status`.
2. Service pings Ollama `/api/tags`.
3. Service checks primary model and cleanup model presence.
4. UI prints diagnostics in process log.

## Constraints and Guardrails
- Batch size hard limit: 25
- Parallel active batches: 2
- API default timeout: 15s
- Translation endpoints timeout: 180s
- Review save validates translated answer count equals source answer count
- Batch status outcomes:
  - `processing`
  - `completed`
  - `completed_with_errors`

- `source`, `questionId`, `locale`, `status`

  - `question`
  - `answers[]` (grade + text)

  - `domainLabel`
  - `rationale`

  - `model`
  - `cleanupModel`
  - `cleanupApplied`
  - `cleanupError`
  - `batchId`
  - `translatedBy`
- `reviewedBy`, `reviewedAt`, `createdAt`, `updatedAt`

async function api(method,path,body,baseUrl){
  async function ensureToken(forceRefresh){
    if(rootsToken && !forceRefresh) return rootsToken;
    var user=fbAuth.currentUser;
    if(!user) return null;
    rootsToken=await user.getIdToken(!!forceRefresh);
    return rootsToken;
  }

  var token=await ensureToken(false);
  if(!token) throw new Error('Not authenticated. Please sign in again.');

  var base=baseUrl||API_TR;
  var opts={method:method,headers:{'Content-Type':'application/json','Authorization':'Bearer '+token}};
  if(body)opts.body=JSON.stringify(body);
  var res=await fetch(base+path,opts);
  if(res.status===401){
    token=await ensureToken(true);
    if(!token) throw new Error('Session expired. Please sign in again.');
    opts.headers.Authorization='Bearer '+token;
    res=await fetch(base+path,opts);
  }

  var data;
  try{data=await res.json();}
  catch(_){data={};}

  if(res.status===401||res.status===403)throw new Error('Unauthorized ('+res.status+')');
  if(!res.ok)throw new Error((data.error&&data.error.message)||data.message||data.error||('HTTP '+res.status));
  return data;
}

// ─── GET /sessions ────────────────────────────────────────────
/**
 * List all sessions across child, parent, teacher collections.
 * Returns key fields for each session (not full answers).
 * Query params: ?status=completed&limit=100
 */
router.get('/sessions', ...systemOnly, async (req: RootsAuthRequest, res) => {
  try {
    const statusFilter = req.query.status as string | undefined;
    const limit = Math.min(parseInt(req.query.limit as string) || 200, 500);

    const collections = [
      { name: 'roots_sessions', role: 'child' },
      { name: 'roots_parent_sessions', role: 'parent' },
      { name: 'roots_teacher_sessions', role: 'teacher' },
      { name: 'anonymous_sessions', role: 'anonymous' },
    ];

    const allSessions: any[] = [];

    for (const col of collections) {
      try {
        const snap = await db.collection(col.name).get();
        for (const doc of snap.docs) {
          const d = doc.data();
          // Normalize v2 'in_progress' → 'active' for consistent display
          const normalizedStatus = d.status === 'in_progress' ? 'active' : (d.status || 'unknown');
          if (statusFilter && normalizedStatus !== statusFilter) continue;

          const startedAt = d.startedAt?.toDate?.() ?? d.startedAt ?? null;
          const completedAt = d.completedAt?.toDate?.() ?? d.completedAt ?? null;

          // Compute PRAI on-the-fly for legacy sessions that lack it
          let praiObj: any = d.prai || null;
          let hasPrai = !!(d.prai);
          if (!hasPrai && d.status === 'completed' && Array.isArray(d.questionsAnswered) && d.questionsAnswered.length > 0) {
            try {
              const catEngine = new CATEngine();
              const grades = d.questionsAnswered.map((a: any) => a.grade).filter(Boolean) as AnswerGrade[];
              const times = d.questionsAnswered.map((a: any) => a.responseTimeMs).filter((t: any) => typeof t === 'number' && t > 0);
              if (grades.length > 0) {
                const allAnswered = d.questionsAnswered.map((a: any) => ({ questionId: a.questionId, grade: a.grade as AnswerGrade }));
                const cFlags = catEngine.runConsistencyChecks(allAnswered);
                const computed = catEngine.computePRAI(grades, times, cFlags);
                praiObj = computed;
                hasPrai = true;
              }
            } catch (_praiErr) {
              // Non-fatal — leave praiObj null
            }
          }
          const praiScore = praiObj?.score ?? null;

          // Build compact posteriors: { domain: { mean, alpha, beta } }
          let posteriorScores: Record<string, { mean: number; alpha: number; beta: number }> | null = null;
          if (d.posteriors && typeof d.posteriors === 'object') {
            posteriorScores = {};
            for (const [domain, post] of Object.entries(d.posteriors)) {
              const p = post as any;
              if (p && typeof p.mean === 'number') {
                posteriorScores[domain] = {
                  mean: parseFloat((p.mean * 100).toFixed(2)),
                  alpha: parseFloat((p.alpha ?? 2).toFixed(2)),
                  beta: parseFloat((p.beta ?? 2).toFixed(2)),
                };
              }
            }
            if (Object.keys(posteriorScores).length === 0) posteriorScores = null;
          }

          allSessions.push({
            sessionId: d.sessionId || doc.id,
            role: d.role || d.assessmentType || col.role,
            status: normalizedStatus,
            ageBand: d.ageBand || null,
            schoolId: d.schoolId || null,
            rootsId: d.rootsId || null,
            questionsAnswered: Array.isArray(d.questionsAnswered) ? d.questionsAnswered.length : 0,
            domainsCovered: d.domainsCovered || d.probedAxes || [],
            engineVersion: d.engineVersion || d.engine_version || null,
            terminationReason: d.terminationReason || null,
            startedAt: startedAt ? new Date(startedAt).toISOString() : null,
            completedAt: completedAt ? new Date(completedAt).toISOString() : null,
            hasPrai,
            praiScore,
            prai: praiObj,
            valenceFlags: d.valenceFlags || praiObj?.flags || [],
            consistencyFlags: Array.isArray(d.consistencyFlags)
              ? d.consistencyFlags.map((cf: any) => ({
                  group: cf.group, axis: cf.axis, cv: cf.cv,
                  label: cf.label, modifier: cf.modifier, warning: cf.warning,
                }))
              : [],
            learningModeActive: !!d.learningModeActive,
            finalProfileId: d.finalProfileId || null,
            // v2 anonymous session extras
            email: d.email || null,
            name: d.name || null,
            inviteId: d.inviteId || d.inviteCode || null,
            scoreArtifact: d.scoreArtifact || null,
            posteriorScores,
            collection: col.name,
          });
        }
      } catch (colErr: any) {
        // Skip collections that don't exist yet
        console.warn(`Session list: skipped ${col.name}: ${colErr.message}`);
      }
    }

    // Attach generated-question counts via a single lightweight projection query.
    try {
      const genSnap = await db.collection('roots_generated_questions')
        .select('sessionId', 'status')
        .get();
      const genTotalMap = new Map<string, number>();
      const genPendingMap = new Map<string, number>();
      for (const doc of genSnap.docs) {
        const { sessionId: sid, status: qStatus } = doc.data() as any;
        if (!sid) continue;
        genTotalMap.set(sid, (genTotalMap.get(sid) ?? 0) + 1);
        if (qStatus === 'engine_review') {
          genPendingMap.set(sid, (genPendingMap.get(sid) ?? 0) + 1);
        }
      }
      for (const sess of allSessions) {
        sess.generatedQuestionsCount = genTotalMap.get(sess.sessionId) ?? 0;
        sess.generatedQuestionsPending = genPendingMap.get(sess.sessionId) ?? 0;
      }
    } catch (_) { /* non-fatal — counts default to 0 */ }

    // Sort: completed first, then by startedAt descending
    allSessions.sort((a, b) => {
      if (a.status === 'completed' && b.status !== 'completed') return -1;
      if (b.status === 'completed' && a.status !== 'completed') return 1;
      const ta = a.startedAt ? new Date(a.startedAt).getTime() : 0;
      const tb = b.startedAt ? new Date(b.startedAt).getTime() : 0;
      return tb - ta;
    });

    const limited = allSessions.slice(0, limit);

    res.json({
      data: limited,
      meta: {
        total: allSessions.length,
        returned: limited.length,
        byStatus: {
          completed: allSessions.filter(s => s.status === 'completed').length,
          active: allSessions.filter(s => s.status === 'active' || s.status === 'in_progress').length,
          paused: allSessions.filter(s => s.status === 'paused').length,
          expired: allSessions.filter(s => s.status === 'expired').length,
        },
        byRole: {
          child: allSessions.filter(s => s.role === 'child').length,
          parent: allSessions.filter(s => s.role === 'parent' || s.role === 'anonymous').length,
          teacher: allSessions.filter(s => s.role === 'teacher').length,
          anonymous: allSessions.filter(s => s.collection === 'anonymous_sessions').length,
        },
      },
    });
  } catch (err: any) {
    rootsErrorResponse(res, err);
  }
});

// ─── GET /generated-questions ─────────────────────────────────
/**
 * List all engine-generated questions pending admin review.
 */
router.get('/generated-questions', ...systemOnly, async (req: RootsAuthRequest, res) => {
  try {
    const status = req.query.status as string || 'engine_review';
    let questions;
    if (status === 'all') {
      const stats = await questionGenerator.getStats();
      questions = await questionGenerator.getPendingReview(); // returns pending; stats has the counts
      return res.json({ data: questions, meta: stats });
    }
    questions = await questionGenerator.getPendingReview();
    const stats = await questionGenerator.getStats();

    // Enrich with similarity analysis
    if (questions.length > 0) {
      try {
        const poolSnap = await rootsCollections.questions().where('status', 'in', ['active', 'review']).get();
        const pool = poolSnap.docs.map(d => ({ id: d.id, ...d.data() } as any));
        const reports = computeSimilarity(questions as any[], pool);
        const reportMap = new Map(reports.map(r => [r.questionId, r]));
        for (const q of questions) {
          const simReport = reportMap.get(q.id);
          if (simReport) (q as any).similarity = simReport;
        }
      } catch (_) { /* non-fatal */ }
    }

    res.json({ data: questions, meta: stats });
  } catch (err: any) {
    rootsErrorResponse(res, err);
  }
});

// ─── POST /generated-questions/retroactive ────────────────────
/**
 * Retroactively generate follow-up questions for ALL completed sessions
 * that don't already have generated questions. Runs the question generator
 * on each session's score artifact (computed on-the-fly if needed).
 */
router.post('/generated-questions/retroactive', ...systemOnly, async (req: RootsAuthRequest, res) => {
  try {
    const collections = ['roots_parent_sessions', 'roots_teacher_sessions', 'roots_sessions', 'anonymous_sessions'];
    let processed = 0;
    let generated = 0;
    let skipped = 0;
    const errors: string[] = [];

    for (const colName of collections) {
      try {
        const snap = await db.collection(colName).get();
        for (const doc of snap.docs) {
          const sess = doc.data();
          if (sess.status !== 'completed') { skipped++; continue; }
          if (!sess.questionsAnswered?.length) { skipped++; continue; }

          const sessionId = sess.sessionId || doc.id;

          // Check if questions already exist for this session
          const existing = await db.collection('roots_generated_questions')
            .where('sessionId', '==', sessionId)
            .limit(1)
            .get();
          if (!existing.empty) { skipped++; continue; }

          // Get or compute score artifact
          let scoreArtifact: any = null;

          // v2 anonymous sessions have scoreArtifact embedded in session doc
          if (sess.scoreArtifact) {
            scoreArtifact = sess.scoreArtifact;
          }

          if (!scoreArtifact) {
            try {
              const saSnap = await db.collection('roots_score_artifacts')
                .where('sessionId', '==', sessionId)
                .limit(1)
                .get();
              if (!saSnap.empty) {
                scoreArtifact = saSnap.docs[0].data();
              }
            } catch (_) { /* skip */ }
          }

          if (!scoreArtifact) {
            // Compute on-the-fly
            try {
              const role = sess.parentId ? 'parent' : sess.teacherId ? 'teacher' : 'child';
              const answers = sess.questionsAnswered.map((a: any) => {
                const domains = Object.keys(a.axesScores || {});
                if (domains.length === 0) {
                  return [{ questionId: a.questionId, domain: a.domain || a.axis || 'cognitive_ability', grade: (a.grade || 'C') as any, response_time_ms: a.responseTimeMs || 0 }];
                }
                return domains.map((domain: string) => ({ questionId: a.questionId, domain, grade: (a.grade || 'C') as any, response_time_ms: a.responseTimeMs || 0 }));
              }).flat();
              if (answers.length > 0) {
                scoreArtifact = computeScores({ sessionId, participantId: sess.rootsId || 'unknown', role: role as any, answers });
              }
            } catch (_) { /* skip */ }
          }

          if (!scoreArtifact) { skipped++; continue; }

          try {
            const qs = await questionGenerator.generateQuestions(scoreArtifact, sessionId, 4);
            generated += qs.length;
            processed++;
          } catch (e: any) {
            errors.push(`${sessionId}: ${e.message}`);
          }
        }
      } catch (colErr: any) {
        console.warn(`Retroactive gen: skipped ${colName}: ${colErr.message}`);
      }
    }

    res.json({
      data: { processed, generated, skipped, errors: errors.slice(0, 10) },
      meta: { message: `Retroactively generated ${generated} questions across ${processed} sessions.` },
    });
  } catch (err: any) {
    rootsErrorResponse(res, err);
  }
});

// ─── GET /question-pools ──────────────────────────────────────
/**
 * Returns summary of ALL question pools:
 * - Child (CAT) — from Firestore roots_questions
 * - Parent (100 in-code) — PARENT_QUESTIONS_CORE + EXTENDED
 * - Teacher (100 in-code) — TEACHER_QUESTIONS_CORE + EXTENDED
 */

  try {
    // ── Child pool from Firestore ──

    for (const id of PARENT_CORE_IDS) {

    }

    // ── Teacher pool from code ──
    const TEACHER_CORE_IDS = Array.from({ length: 12 }, (_, i) => `TA${String(i + 1).padStart(2, '0')}`);
    const teacherDomains: Record<string, { count: number; ids: string[]; active: number }> = {};

      teacherDomains[dom].count++;

    for (const q of TEACHER_QUESTIONS_EXTENDED) {

    rootsErrorResponse(res, err);

// ═══════════════════════════════════════════════════════════════
//  POST /simulate-sessions
//
//  Generates realistic synthetic assessment sessions using the GRM
//  (Graded Response Model, Samejima 1969) and writes them to Firestore
//  through the full scoring/report pipeline via onSessionCompleted().
//
//  Each session gets per-domain theta values drawn from the requested
//  profile, producing genuinely varied domain scores (not all-S).
//  Sessions are tagged isSimulated:true for auditability.
//
//  Body: { role, count, profile, seed? }
//    role:    'parent' | 'teacher'
//    count:   1–10
//    profile: 'realistic' | 'high' | 'mixed' | 'low'
//    seed:    optional integer for reproducible output
// ═══════════════════════════════════════════════════════════════

router.post('/simulate-sessions', ...systemOnly, async (req: RootsAuthRequest, res) => {
  try {
    const { role = 'parent', count = 3, profile = 'realistic', seed } = req.body as {
      role?: string; count?: number; profile?: string; seed?: number;
    };
    if (!['parent', 'teacher'].includes(role)) {
      return res.status(400).json({ error: 'role must be parent or teacher' });
    }
    const n = Math.min(10, Math.max(1, Number(count) || 3));

    // ── Seeded Mulberry32 RNG (deterministic when seed provided) ──
    let _rngState = ((seed ?? Math.floor(Math.random() * 0xFFFFFFFF)) ^ 0xDEADBEEF) >>> 0;
    const rng = () => {
      _rngState = (_rngState + 0x6D2B79F5) >>> 0;
      let t = Math.imul(_rngState ^ (_rngState >>> 15), 1 | _rngState);
      t = (t + Math.imul(t ^ (t >>> 7), 61 | t)) >>> 0;
      return ((t ^ (t >>> 14)) >>> 0) / 4294967296;
    };

    // ── GRM grade sampler (Samejima 1969, a=4) ──
    const GRM_A = 4.0;
    const GRM_B = [0.25, 0.45, 0.65, 0.82]; // difficulty thresholds C/B/A/S
    const GRADE_LETTERS = ['D', 'C', 'B', 'A', 'S'] as const;
    const GRADE_SCORES: Record<string, number> = { S: 5, A: 4, B: 3, C: 2, D: 1, F: 0 };
    const GRADE_AXES_NORM: Record<string, number> = { S: 1.0, A: 0.8, B: 0.6, C: 0.4, D: 0.2, F: 0.0 };
    const sigmoid = (x: number) => 1 / (1 + Math.exp(-x));

    function sampleGrade(theta: number): typeof GRADE_LETTERS[number] {
      const cumP = GRM_B.map(b => sigmoid(GRM_A * (theta - b)));
      let r = rng();
      if (r < 1 - cumP[0]) return 'D';
      r -= (1 - cumP[0]);
      if (r < cumP[0] - cumP[1]) return 'C';
      r -= (cumP[0] - cumP[1]);
      if (r < cumP[1] - cumP[2]) return 'B';
      r -= (cumP[1] - cumP[2]);
      if (r < cumP[2] - cumP[3]) return 'A';
      return 'S';
    }

    // ── Theta sampler per profile ──
    // Beta sampling via Box-Muller approximation: Beta(a,b) ≈ truncated via rejection
    function sampleBeta(a: number, b: number): number {
      // Johnk's method (exact for small a,b)
      for (let attempt = 0; attempt < 50; attempt++) {
        const u = Math.pow(rng(), 1 / a);
        const v = Math.pow(rng(), 1 / b);
        if (u + v <= 1) return Math.max(0.05, Math.min(0.95, u / (u + v)));
      }
      return a / (a + b); // fallback to mean
    }

    function sampleTheta(profileName: string): number {
      switch (profileName) {
        case 'high':     return sampleBeta(4.0, 1.5); // mean ≈ 0.73, skewed high
        case 'low':      return sampleBeta(1.5, 4.0); // mean ≈ 0.27, skewed low
        case 'mixed':    return 0.15 + rng() * 0.70;  // uniform [0.15, 0.85]
        default:         return sampleBeta(2.5, 2.5); // 'realistic': centred ~0.50, spread
      }
    }

    // ── Response time sampler (lognormal, typical 3–25s) ──
    function sampleResponseTimeMs(): number {
      // Box-Muller lognormal: μ=2.1 (≈8s), σ=0.5
      const u1 = Math.max(1e-10, rng()), u2 = rng();
      const z = Math.sqrt(-2 * Math.log(u1)) * Math.cos(2 * Math.PI * u2);
      const ms = Math.exp(2.1 + 0.5 * z) * 1000;
      return Math.round(Math.min(60000, Math.max(1200, ms)));
    }

    // ── Question pool ──
    const { v4: uuidv4 } = await import('uuid');
    const allParent: Array<{ id: string; domain: string; text: string; answers: Array<{ grade: string; text: string; axes: Record<string, number> }> }> =
      [...PARENT_QUESTIONS_CORE, ...PARENT_QUESTIONS_EXTENDED].map(q => ({
        id: q.id, domain: q.domain as string, text: q.text,
        answers: q.answers.map(a => ({ grade: a.grade as string, text: a.text, axes: a.axes as Record<string, number> })),
      }));
    const allTeacher: Array<{ id: string; domain: string; text: string; answers: Array<{ grade: string; text: string; axes: Record<string, number> }> }> =
      [...TEACHER_QUESTIONS_CORE, ...TEACHER_QUESTIONS_EXTENDED].map((q: any) => ({
        id: q.id, domain: q.domain as string, text: q.text,
        answers: q.answers.map((a: any) => ({ grade: a.grade as string, text: a.text, axes: a.axes as Record<string, number> || {} })),
      }));
    const pool = role === 'parent' ? allParent : allTeacher;

    // Domain lists
    const PARENT_DOMAINS_SIM = ['home_environment','emotional_availability','expectation_calibration',
      'growth_orientation','resilience_modeling','cultural_exposure','communication_quality','aspiration_alignment'];
    const TEACHER_DOMAINS_SIM = ['pedagogical_curiosity','student_empathy','resilience_modeling',
      'systems_thinking','character_consistency','creative_intelligence','somatic_presence','meaning_transfer'];
    const domains = role === 'parent' ? PARENT_DOMAINS_SIM : TEACHER_DOMAINS_SIM;

    // ── Build one question set covering all domains (2 per domain = 16q, trim to 15) ──
    function buildQuestionSet(): typeof pool {
      const byDomain: Record<string, typeof pool> = {};
      for (const d of domains) byDomain[d] = [];
      for (const q of pool) {
        const dom = q.domain as string;
        if (byDomain[dom]) byDomain[dom].push(q as any);
      }
      const selected: typeof pool = [];
      // 2 questions per domain, shuffled within domain
      for (const d of domains) {
        const dPool = byDomain[d] || [];
        // Fisher-Yates shuffle
        for (let i = dPool.length - 1; i > 0; i--) {
          const j = Math.floor(rng() * (i + 1));
          [dPool[i], dPool[j]] = [dPool[j], dPool[i]];
        }
        selected.push(...dPool.slice(0, 2));
      }
      return selected.slice(0, 15); // cap at 15 questions
    }

    // ── Conjugate Beta update ──
    function betaUpdate(prior: { alpha: number; beta: number }, axesNorm: number) {
      return { alpha: prior.alpha + axesNorm, beta: prior.beta + (1 - axesNorm) };
    }
    function betaMean(p: { alpha: number; beta: number }) { return p.alpha / (p.alpha + p.beta); }
    function betaConf(p: { alpha: number; beta: number }) {
      const v = (p.alpha * p.beta) / ((p.alpha + p.beta) ** 2 * (p.alpha + p.beta + 1));
      return Math.max(0, 1 - 2 * Math.sqrt(v));
    }

    // ── Run one simulation ──
    const createdSessions: { sessionId: string; role: string; dominantDomain: string; overallMean: number; theta: Record<string, number> }[] = [];

    const { FieldValue: FV, Timestamp: FBTimestamp } = await import('firebase-admin/firestore');

    for (let i = 0; i < n; i++) {
      // Sample theta profile for this respondent
      const theta: Record<string, number> = {};
      for (const d of domains) theta[d] = sampleTheta(profile);

      const sessionId = uuidv4();
      const simRootsId = `sim_${role[0]}_${sessionId.slice(0, 8)}`;
      const sessionCol = role === 'parent' ? 'roots_parent_sessions' : 'roots_teacher_sessions';

      // Starting posteriors: Beta(1,1) uniform prior
      const posteriors: Record<string, { alpha: number; beta: number; mean: number }> = {};
      for (const d of domains) posteriors[d] = { alpha: 1, beta: 1, mean: 0.5 };

      // Build question set
      const questions = buildQuestionSet();
      const questionsAnswered: any[] = [];
      const startedAt = new Date(Date.now() - (15 + Math.floor(rng() * 10)) * 60000); // 15-25 min ago

      for (const q of questions) {
        const dom = q.domain as string;
        const domainTheta = theta[dom] ?? theta[domains[0]];
        const grade = sampleGrade(domainTheta);
        const axesNorm = GRADE_AXES_NORM[grade] ?? 0.5;
        const responseTimeMs = sampleResponseTimeMs();

        // Build axesScores from this answer's axes
        const axesRaw = (q.answers.find((a: any) => a.grade === grade) as any)?.axes || { [dom]: GRADE_SCORES[grade] };
        const axesScores: Record<string, number> = {};
        for (const [ax, rawScore] of Object.entries(axesRaw as Record<string, number>)) {
          axesScores[ax] = rawScore;
        }

        const posteriorBefore = { ...posteriors };
        // Update all axes touched by this answer
        const touchedDoms = Object.keys(axesRaw);
        for (const ax of touchedDoms) {
          const axNorm = (axesRaw[ax] as number) / 5; // normalize 1-5 → 0-1
          const updated = betaUpdate(posteriors[ax] || { alpha: 1, beta: 1 }, axNorm);
          posteriors[ax] = { ...updated, mean: betaMean(updated) };
        }

        const answerIdx = q.answers.findIndex((a: any) => a.grade === grade);
        questionsAnswered.push({
          questionId:     q.id,
          domain:         dom,
          grade,
          axesScores,
          answerIndex:    answerIdx >= 0 ? answerIdx : 0,
          responseTimeMs,
          posteriorBefore,
          posteriorAfter: { ...posteriors },
          timestamp:      new Date(startedAt.getTime() + questionsAnswered.length * responseTimeMs).toISOString(),
        });
      }

      // Compute PRAI
      const { CATEngine: CATEng } = await import('../services/cat-engine.service.js');
      const catEng = new CATEng();
      const grades = questionsAnswered.map((a: any) => a.grade as AnswerGrade);
      const times  = questionsAnswered.map((a: any) => a.responseTimeMs as number);
      const prai   = catEng.computePRAI(grades, times, []);

      // Dominant domain (highest posterior mean)
      const dominantDomain = domains.reduce((best, d) =>
        (posteriors[d]?.mean ?? 0) > (posteriors[best]?.mean ?? 0) ? d : best, domains[0]);
      const overallMean = domains.reduce((s, d) => s + (posteriors[d]?.mean ?? 0.5), 0) / domains.length;

      // Write session document
      const sessionDoc: Record<string, any> = {
        sessionId,
        [role === 'parent' ? 'parentId' : 'teacherId']: simRootsId,
        childId:           role === 'parent' ? `sim_child_${sessionId.slice(0, 8)}` : undefined,
        ownerUid:          `sim_uid_${sessionId.slice(0, 8)}`,
        schoolId:          'simulation',
        consentGiven:      true,
        startedAt:         FBTimestamp.fromDate(startedAt),
        completedAt:       FV.serverTimestamp(),
        status:            'completed',
        questionsAnswered,
        posteriors,
        valenceFlags:      prai.flags,
        consistencyFlags:  [],
        prai,
        engineVersion:     ENGINE_VERSION,
        finalProfileId:    null,
        isSimulated:       true,
        simulationProfile: profile,
        simulationTheta:   theta,
        _simulatedAt:      FV.serverTimestamp(),
      };
      if (role === 'teacher') delete sessionDoc.childId;

      await db.collection(sessionCol).doc(sessionId).set(sessionDoc);

      // Run full scoring pipeline (fire-and-forget but await for data accuracy)
      try {
        await onSessionCompleted({
          sessionId,
          rootsId: simRootsId,
          role: role as 'parent' | 'teacher',
          answers: questionsAnswered.map((a: any) => ({
            questionId:     a.questionId,
            domain:         a.domain,
            grade:          a.grade,
            axesScores:     a.axesScores,
            responseTimeMs: a.responseTimeMs,
          })),
          posteriors,
        });
      } catch (scoreErr: any) {
        console.warn(`[Simulate] Scoring failed for ${sessionId} (non-fatal):`, scoreErr?.message);
      }

      createdSessions.push({ sessionId, role, dominantDomain, overallMean: parseFloat(overallMean.toFixed(3)), theta });
    }

    res.json({
      created: createdSessions.length,
      profile,
      role,
      sessions: createdSessions,
      note: 'Sessions tagged isSimulated:true. Visible immediately in admin portal Sessions tab.',
    });
  } catch (err: any) {
    rootsErrorResponse(res, err);
  }
});

// ═══════════════════════════════════════════════════════════════
//  GET /question-bank-full  — complete question text map for admin UI
//  Returns {id → {text, domain, role, answers:[{grade,text,axes}]}} for
//  ALL questions: child (Firestore) + parent/teacher (in-code).
//  Used by the admin portal to hydrate question IDs to readable text.
// ═══════════════════════════════════════════════════════════════

router.get('/question-bank-full', ...systemOnly, async (_req, res) => {
  try {
    const bank: Record<string, { text: string; domain: string; role: string; answers: Array<{ grade: string; text: string; axes: Record<string, number> }> }> = {};

    // ── Child questions from Firestore ──
    const childSnap = await rootsCollections.questions().limit(1000).get();
    for (const doc of childSnap.docs) {
      const q = doc.data() as any;
      bank[doc.id] = {
        text: q.text || '',
        domain: q.domain || '',

        answers: Array.isArray(q.answers)
          ? q.answers.map((a: any) => ({ grade: a.grade || '', text: a.text || '', axes: a.axes || {} }))
          : [],
      };
      if (q.id && q.id !== doc.id) bank[q.id] = bank[doc.id];
    }

    // ── Parent questions (core PA01–PA12 + extended PA13–PA100) ──
    const allParent = [...PARENT_QUESTIONS_CORE, ...PARENT_QUESTIONS_EXTENDED];
    for (const q of allParent) {
      bank[q.id] = {
        text: q.text,
        domain: q.domain,
        role: 'parent',
        answers: q.answers.map(a => ({ grade: a.grade, text: a.text, axes: (a.axes as Record<string, number>) || {} })),
      };
    }

    // ── Teacher questions (core TA01–TA12 + extended TA13–TA100) ──
    const allTeacher = [...TEACHER_QUESTIONS_CORE, ...TEACHER_QUESTIONS_EXTENDED];
    for (const q of allTeacher) {

        domain: q.domain,

        answers: q.answers.map((a: any) => ({ grade: a.grade, text: a.text, axes: a.axes || {} })),

router.get('/consent-metrics', ...systemOnly, async (_req, res) => {
  try {
    const snap = await db.collection('roots_consent_events').get();
    const events: Record<string, number> = { shown: 0, accepted: 0, declined: 0 };
    const byInvite: Record<string, { shown: number; accepted: number; declined: number }> = {};
    const timeline: { date: string; shown: number; accepted: number; declined: number }[] = [];
    const dailyMap: Record<string, { shown: number; accepted: number; declined: number }> = {};

    for (const doc of snap.docs) {
      const d = doc.data();
      const ev = d.event as string;
      if (events[ev] !== undefined) events[ev]++;

      // By invite
      const code = d.inviteCode || 'unknown';
      if (!byInvite[code]) byInvite[code] = { shown: 0, accepted: 0, declined: 0 };
      if (byInvite[code][ev as keyof typeof byInvite[typeof code]] !== undefined) {
        byInvite[code][ev as keyof typeof byInvite[typeof code]]++;
      }

      // Daily timeline
      const ts = d.timestamp || d.createdAt?.toDate?.()?.toISOString?.() || '';
      const day = ts.substring(0, 10);
      if (day) {
        if (!dailyMap[day]) dailyMap[day] = { shown: 0, accepted: 0, declined: 0 };
        if (dailyMap[day][ev as keyof typeof dailyMap[typeof day]] !== undefined) {
          dailyMap[day][ev as keyof typeof dailyMap[typeof day]]++;
        }
      }
    }

    // Sort timeline by date
    for (const [date, counts] of Object.entries(dailyMap).sort()) {
      timeline.push({ date, ...counts });
    }

    const abandonmentRate = events.shown > 0
      ? ((events.declined / events.shown) * 100).toFixed(1) + '%'
      : '0%';
    const conversionRate = events.shown > 0
      ? ((events.accepted / events.shown) * 100).toFixed(1) + '%'
      : '0%';

    res.json({
      data: {
        totals: events,
        abandonmentRate,
        conversionRate,
        byInvite,
        timeline,
      },
    });
  } catch (err: any) {
    rootsErrorResponse(res, err);
  }
});

function translationDocId(source: QuestionSourceType, questionId: string, locale: TranslationLocale): string {

function sourceHash(q: SourceQuestion): string {
  const normalized = JSON.stringify({
    question: q.question,
    answers: q.answers.map((a) => ({ grade: a.grade, text: a.text })),
    tags: q.tags,
    domain: q.domain,
    rationale: q.rationale ?? null,
  });
  return createHash('sha256').update(normalized).digest('hex');
}

function normalizeAxes(axes: unknown): Record<string, number> {
  if (!axes || typeof axes !== 'object') return {};
  const out: Record<string, number> = {};
  for (const [k, v] of Object.entries(axes as Record<string, unknown>)) {
    if (typeof v === 'number' && Number.isFinite(v)) out[k] = v;
  }
  return out;
}

async function loadTagMemory(locale: TranslationLocale): Promise<Record<string, string>> {

async function saveTagMemory(locale: TranslationLocale, updates: Record<string, string>): Promise<void> {
  if (Object.keys(updates).length === 0) return;
  const patch: Record<string, unknown> = {};
  for (const [k, v] of Object.entries(updates)) {
    patch[`terms.${k}`] = v;

  patch.updatedAt = FieldValue.serverTimestamp();
  await db.collection(MEMORY_COLLECTION).doc(locale).set(patch, { merge: true });
}

function extractJsonObject(raw: string): unknown {
  let s = raw.replace(/```(?:json)?/gi, '').trim();
  const start = s.indexOf('{');
  const end = s.lastIndexOf('}');
  if (start === -1 || end === -1 || end < start) {
    throw new Error(`Cleanup model returned no JSON object. raw=${s.slice(0, 300)}`);
  }
  s = s.slice(start, end + 1);
  s = s.replace(/,\s*([}\]])/g, '$1');
  s = s.replace(/[\x00-\x08\x0b\x0c\x0e-\x1f]/g, '');
  return JSON.parse(s);
}

/**
 * Translate a single plain-text string using the local Ollama translation model.
 * The sarvam-translate model is a pure seq2seq translator — it translates input text
 * and returns translated text. We call it once per field.
 */
async function translateText(text: string, locale: TranslationLocale): Promise<string> {
  if (!text || !text.trim()) return '';
  const model = process.env.ROOTS_TRANSLATION_MODEL || 'aya-expanse:8b';
  const baseUrl = process.env.ROOTS_TRANSLATION_OLLAMA_URL || 'http://127.0.0.1:11434';
  const localeName = locale === 'hi' ? 'Hindi' : 'Marathi';

  let response: Response;
  try {
    response = await fetch(`${baseUrl}/api/generate`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        model,
        stream: false,
        prompt: `Translate to ${localeName}: ${text}`,
        options: { temperature: 0 },
      }),
    });
  } catch (err: any) {
    const cause = err?.cause?.message || err?.message || 'unknown network error';
    throw new Error(`Local model connection failed. model=${model}, baseUrl=${baseUrl}, reason=${cause}`);
  }

  if (!response.ok) {
    const body = await response.text();
    throw new Error(`Ollama request failed (${response.status}). model=${model}, baseUrl=${baseUrl}, body=${body.slice(0, 240)}`);
  }

  const data = await response.json() as { response?: string };
  const content = data?.response?.trim();
  if (!content) throw new Error('Local model returned empty content');
  return content;
}

function buildCleanupPrompt(source: SourceQuestion, draft: TranslationPayload, locale: TranslationLocale): string {
  const localeName = locale === 'hi' ? 'Hindi' : 'Marathi';
  return [
    `You are a translation QA editor for ${localeName}.`,
    'Polish the translated text for naturalness and consistency while preserving exact meaning and tone.',
    'Do not change answer ordering, count, or response polarity.',
    'Return ONLY one JSON object with keys: question, answers, tags, domainLabel, rationale.',
    '',
    `Source English question: ${source.question}`,
    `Source English answers: ${JSON.stringify(source.answers.map((a) => a.text))}`,
    `Source English tags: ${JSON.stringify(source.tags)}`,
    `Source English domainLabel: ${domainLabel(source.domain)}`,
    `Source English rationale: ${source.rationale ?? ''}`,
    '',
    `Current ${localeName} draft question: ${draft.question}`,
    `Current ${localeName} draft answers: ${JSON.stringify(draft.answers)}`,
    `Current ${localeName} draft tags: ${JSON.stringify(draft.tags)}`,
    `Current ${localeName} draft domainLabel: ${draft.domainLabel}`,
    `Current ${localeName} draft rationale: ${draft.rationale ?? ''}`,
  ].join('\n');
}

async function runCleanupPass(source: SourceQuestion, draft: TranslationPayload, locale: TranslationLocale): Promise<{ payload: TranslationPayload; applied: boolean; model: string | null; error?: string }> {
  const cleanupModel = (process.env.ROOTS_TRANSLATION_CLEANUP_MODEL || '').trim();
  if (!cleanupModel) {
    return { payload: draft, applied: false, model: null };
  }

  const baseUrl = process.env.ROOTS_TRANSLATION_CLEANUP_OLLAMA_URL || process.env.ROOTS_TRANSLATION_OLLAMA_URL || 'http://127.0.0.1:11434';
  const prompt = buildCleanupPrompt(source, draft, locale);

  try {
    const response = await fetch(`${baseUrl}/api/chat`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        model: cleanupModel,
        stream: false,
        options: { temperature: 0.1 },
        messages: [
          {
            role: 'system',
            content: 'You are a strict JSON editor. Output only JSON.',
          },
          { role: 'user', content: prompt },
        ],
      }),
    });

    if (!response.ok) {
      const body = await response.text();
      throw new Error(`cleanup request failed (${response.status}): ${body.slice(0, 200)}`);
    }

    const data = await response.json() as { message?: { content?: string } };
    const content = data?.message?.content?.trim() || '';
    const payload = TranslationPayloadSchema.parse(extractJsonObject(content));
    if (payload.answers.length !== source.answers.length) {
      throw new Error('cleanup answer count mismatch');
    }

    return { payload, applied: true, model: cleanupModel };
  } catch (err: any) {
    return {
      payload: draft,
      applied: false,
      model: cleanupModel,
      error: err?.message || 'cleanup_failed',
    };
  }
}

/**
 * Translate all fields of a source question and assemble a TranslationPayload.
 * Each field is translated independently — suited for seq2seq translation models.
 */
async function runFieldTranslation(source: SourceQuestion, locale: TranslationLocale): Promise<{
  question: string;
  answers: string[];
  tags: string[];
  domainLabel: string;
  rationale: string;
}> {
  const question = await translateText(source.question, locale);
  const answers = await Promise.all(source.answers.map((a) => translateText(a.text, locale)));
  const tags = await Promise.all(source.tags.map((t) => translateText(t, locale)));
  const dl = await translateText(domainLabel(source.domain), locale);
  const rationale = source.rationale ? await translateText(source.rationale, locale) : '';
  return { question, answers, tags, domainLabel: dl, rationale };
}


async function pingOllama(baseUrl: string): Promise<{ ok: boolean; detail: string; models: string[]; elapsedMs: number }> {
  const started = Date.now();
  try {
    const response = await fetch(`${baseUrl}/api/tags`);
    const elapsedMs = Date.now() - started;
    if (!response.ok) {
      return {
        ok: false,
        detail: `GET /api/tags failed with ${response.status}`,
        models: [],
        elapsedMs,
      };
    }

    const payload = await response.json() as { models?: Array<{ name?: string; model?: string }> };
    const models = Array.isArray(payload.models)
      ? payload.models
        .map((m) => m.name || m.model || '')
        .filter(Boolean)
      : [];

    return {
      ok: true,
      detail: `Ollama reachable (${models.length} model(s) visible)`,
      models,
      elapsedMs,
    };
  } catch (err: any) {
    return {
      ok: false,
      detail: err?.cause?.message || err?.message || 'network error',
      models: [],
      elapsedMs: Date.now() - started,
    };
  }
}

        : 'Cleanup model disabled',
      elapsedMs: 0,
    });

      ready: ping.ok && modelLoaded,
      environment,
      model,
      cleanupModel,
      baseUrl,
      chatEndpoint: `${baseUrl}/api/chat`,
      tagsEndpoint: `${baseUrl}/api/tags`,

      // Fallback when composite indexes are not yet built in Firestore.
      const code = err?.code;
      const message = String(err?.message || '').toLowerCase();
      const isIndexError = code === 9 || code === 'failed-precondition' || message.includes('failed_precondition') || message.includes('requires an index');
      if (!isIndexError) throw err;

      const scanLimit = Math.min(Math.max(limit * 10, 200), 2000);
      const snap = await db.collection(TRANSLATION_COLLECTION).limit(scanLimit).get();
      const rows = snap.docs.map((doc) => ({ id: doc.id, ...doc.data() })) as Array<any>;

      const filtered = rows.filter((row) => {
        if (params.source && row.source !== params.source) return false;
        if (params.locale && row.locale !== params.locale) return false;
        if (params.status && row.status !== params.status) return false;
        return true;
      });

      filtered.sort((a, b) => {
        const at = a.updatedAt?._seconds ? Number(a.updatedAt._seconds) : 0;
        const bt = b.updatedAt?._seconds ? Number(b.updatedAt._seconds) : 0;
        return bt - at;
      });

      return filtered.slice(0, limit);
    }
  }

  async translateSingle(params: {

    return updates;
  }

  async createAndRunBatch(params: {
    source: QuestionSourceType;
    locale: TranslationLocale;
    questionIds: string[];
    actor: string;
  }) {
    const uniqueIds = Array.from(new Set(params.questionIds.filter(Boolean)));
    if (uniqueIds.length === 0) throw new Error('At least one questionId is required');
    if (uniqueIds.length > 25) throw new Error('Batch limit exceeded: max 25 questions per batch');

    const active = await db.collection(BATCH_COLLECTION).where('status', 'in', ['queued', 'processing']).get();
    if (active.size >= 2) {
      throw new Error('Parallel batch limit reached: only 2 translation batches can run at once');
    }

    const batchRef = db.collection(BATCH_COLLECTION).doc();
    await batchRef.set({
      source: params.source,
      locale: params.locale,
      questionIds: uniqueIds,
      total: uniqueIds.length,
      success: 0,
      failed: 0,
      status: 'processing',
      createdBy: params.actor,
      createdAt: FieldValue.serverTimestamp(),
      startedAt: FieldValue.serverTimestamp(),
      updatedAt: FieldValue.serverTimestamp(),
      errors: [],
    });

    let success = 0;
    let failed = 0;
    const errors: Array<{ questionId: string; error: string }> = [];

    for (const questionId of uniqueIds) {
      try {
        await this.translateSingle({
          source: params.source,
          questionId,
          locale: params.locale,
          actor: params.actor,
          batchId: batchRef.id,
        });
        success += 1;
      } catch (err: any) {
        failed += 1;
        errors.push({ questionId, error: err?.message || 'Unknown error' });
      }

      await batchRef.set(
        {
          success,
          failed,
          updatedAt: FieldValue.serverTimestamp(),
          errors: errors.slice(0, 50),
        },
        { merge: true },
      );
    }

    await batchRef.set(
      {
        status: failed > 0 ? 'completed_with_errors' : 'completed',
        completedAt: FieldValue.serverTimestamp(),
        updatedAt: FieldValue.serverTimestamp(),
      },
      { merge: true },
    );

    return { batchId: batchRef.id, total: uniqueIds.length, success, failed, errors };
  }

  async listBatches(limit = 50) {
    const snap = await db.collection(BATCH_COLLECTION).orderBy('createdAt', 'desc').limit(Math.min(Math.max(limit, 1), 200)).get();
    return snap.docs.map((doc) => ({ batchId: doc.id, ...doc.data() }));
  }

  async getBatch(batchId: string) {
    const doc = await db.collection(BATCH_COLLECTION).doc(batchId).get();
    if (!doc.exists) return null;
    return { batchId: doc.id, ...doc.data() };
  }

 *   and its SHA-256 hash must appear in ROOTS_ENTERPRISE_KEY_HASHES env var

 *   is open for development convenience but logs a warning.
 * System calls bypass key-value validation (internal server-to-server).
 */
export function requireEnterpriseKey(
  req: RootsAuthRequest,
  res: Response,
  next: NextFunction,

  if (enterpriseKey) req.enterpriseKey = enterpriseKey;

  // Role gate
  if (req.rootsRole !== 'enterprise' && req.rootsRole !== 'system') {

        message: 'Enterprise API key required for intelligence endpoints',
        status: 403,
      },
    });
    return;
  }

  // System calls are trusted server-to-server — skip key-value check.
  if (req.rootsRole === 'system') {
    next();
    return;
  }

  // Enterprise: key header must be present.
  if (!enterpriseKey) {
    res.status(403).json({
      error: {
        code: 'ENTERPRISE_KEY_MISSING',
        message: 'X-Roots-Enterprise-Key header is required for enterprise endpoints',
        status: 403,
      },
    });
    return;
  }

  // Validate key hash against known allowlist.
  const knownHashes = (process.env.ROOTS_ENTERPRISE_KEY_HASHES ?? '').split(',').filter(Boolean);
  if (knownHashes.length === 0) {
    if (process.env.NODE_ENV === 'production') {
      // Hard block in production — misconfiguration must not silently permit access.
      console.error('[requireEnterpriseKey] ROOTS_ENTERPRISE_KEY_HASHES is not set in production. Blocking request.');
      res.status(503).json({

          message: 'Enterprise API is not available at this time',
          status: 503,

    console.warn('[requireEnterpriseKey] ROOTS_ENTERPRISE_KEY_HASHES is not set; key-value check skipped. This must be set in production.');

      error: {
        code: 'ENTERPRISE_KEY_INVALID',
        message: 'Invalid enterprise API key',
        status: 403,
      },
    });
